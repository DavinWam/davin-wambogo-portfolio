Silent Codes

(2023)

Roles: Game Design, Graphics Programming, UI Design

An interactive two-player puzzle game developed over a two-week period, blending traditional escape room elements with digital puzzle-solving. Player 1 navigates a memory puzzle minigame to extract crucial code pieces, while Player 2 uses a physical cipher to decode the information. The game intensifies with the presence of a Boss character, forcing both players to remain quiet and avoid detection.

What I Did

  • Wrote a CRT shader in HLSL for visual effects.
  • Created the minigame for obtaining code pieces.
  • Came up with the busy work task for making the boss think you are working.

Highlight of the Project

For me this project was definitely a about implementing the CRT effect it gave me the chance to start learning HLSL, and see how to apply shaders through screenspace, target parts of the renderpipeline and more. Which became profoundly helpful in later projects, particularly as when I took a dive into Stylized PBR with a another one of my games Baby Steps.
