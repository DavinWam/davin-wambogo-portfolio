Silent Codes

(2023)

Roles: Game Design, Graphics Programming, UI Design

An interactive two-player puzzle game developed over a two-week period, blending traditional escape room elements with digital puzzle-solving. Player 1 navigates a memory puzzle minigame to extract crucial code pieces, while Player 2 uses a physical cipher to decode the information. The game intensifies with the presence of a Boss character, forcing both players to remain quiet and avoid detection.

  • Wrote a CRT shader in HLSL simulating scanlines, lens distortion, screen blinking, and other effects to enhance the game's thematic setting.
  • Designed the pretend-to-work gameplay loop, and the minigame for obtaining code pieces.

Things I Learned

  • Writing shader code directly in HLSL.
  • Hooking into Unity's rendering system to apply materials across multiple UI elements as a texture

Highlight of the Project

Working on Silent Codes offered me a profound opportunity to demystify Unity's rendering process. Interacting directly with how elements are drawn to the screen opened up new avenues for creative and efficient problem solving. I'm especially proud of implementing the CRT effects, which involved writing shader code and integrating it seamlessly within Unity's rendering system. This hands-on experience not only enhanced the game's visual appeal but also deepened my understanding of graphical programming and asset optimization. This project has empowered me with the knowledge and confidence to intervene and extend the engine's capabilities where necessary, ensuring that I can always align the technology with my creative vision in future projects.
