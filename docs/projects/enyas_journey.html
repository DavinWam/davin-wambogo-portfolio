Enya's Journey

(2023)

Roles: Game & Level Designer, Programmer, Animator, Special Effects Artist, and UI Designer

A 2D RPG that blends pixel-art characters with modeled environments, created for a Game Programming school project over two months by a five-person team. A fantasy Adventure built around fire, in an unstable world. One where a hero is chosen periodically to refuel the worlds flame, lest the monster population increase to apocalyptic levels. My teammates contributed music, narrative writing, sprite creation, pixel art animation, and story programming.

What I Did

  • Authored the class document for programming and the Game Design document.
  • programmed the battle system, covering stats, turn flow, spells, enemy encounters, etc.
  • Designed the rpg systems from class structure, spells, items,.
  • Designed and decorated the entire map, and created textures for terrain and water.
  • All custom shaders were created by me in Shader Graph.
  • All spell effects and fire effects were made by me in Visual Effects Graph
  • Implemented and designed all the UI, except for the dialogue system.

Highlight of the Project

Working on Enya's Journey was signifigant to me as an avid fan of RPGs. This project was a testament to my growth as a developer, where time spent learning rather than making was less than previous projects. I was truly immersed and excited in developing Enya's Journey for those two month, partly because I had to, if it was going to be finished in time.

The design I had drawn, provided many systems that needed to be created and I learned throughout how that compounds on itself. The core system, the battle programming consumed a month and a third of development time.This project not only allowed me to channel my love for RPGs into a tangible form but also represented a major leap forward in my capability to independently develop a game from scratch.
