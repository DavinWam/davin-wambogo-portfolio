Analog Horror Project

(2024)

Roles: Techinal Art, Level Design, Graphics Programming

A collaborative effort in creating an analog horror experience, inspired by the unsettling and eerie nature of analog media. This project was developed in collaboration with Andrei Davydov, Lois Kim, and Astrid Da Costa, over a week. I was responsible for the design of the forest made the terrain, placed the assets, set the post processing, and implemented the shader UI, and controls.

  • Implemented PostProcessing Effects shader to emulate a analog camera in hlsl
  • Experimented with Displacement maps for terrain, which is not default functionality for Unity

Things I Learned

  • Packing data into color channels that I could use to edit the height map local to the texture
