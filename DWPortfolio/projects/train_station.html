Analog Horror Project

(2024)

Roles: Technical Art, Level Design, Graphics Programming

This project is a collaborative effort in creating an analog horror experience set in a train station. The liminal space setting enhances the eerie and unsettling nature of analog horror, making the viewer question the reality of the environment. I was responsible for designing the forest and train station terrains, placing assets, setting up post-processing effects, and implementing shader UI and controls.

  • Implemented post-processing effects shader to emulate an analog camera in HLSL.
  • Experimented with displacement maps for terrain, adding an extra layer of detail not typically available in Unity.

Things I Learned

  • Packing data into color channels to manipulate the height map localized to the texture.
